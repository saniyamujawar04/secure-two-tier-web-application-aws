# Secure Two-Tier Web Application on AWS

## Project Overview

This project demonstrates the deployment of a secure two-tier web application using AWS cloud services.

The application is hosted on Amazon EC2 and securely communicates with Amazon RDS for database operations. Amazon S3 is used for file storage, AWS IAM manages access permissions, and AWS Secrets Manager securely stores sensitive credentials.
